# Marketing-campaign-dashboard
Excel dashboard analyzing marketing campaign performance with KPIs, ROI, CPA, pivot tables and slicers.

# Tools Used

# Microsoft Excel

Pivot Tables & Pivot Charts
KPI Cards
Slicers
Custom Formatting & Formulas
Dynamic chart formatting
Data cleaning & transformation


# Key Features

Dynamic KPI Cards (ROI %, CPA, Conversions)
Channel wise & Campaign wise performance visualizations
Interactive slicers (Month, Location, Campaign)
Clean and business ready dashboard layout

# Insights

Some of the insights the dashboard helps identify:

Best performing campaign based on ROI %

Campaigns generating highest conversions

Average acquisition cost of different channels

Audience segments with highest conversions

Trends over time in campaign efficiency

Top location with highest conversions
